# 极简调酒卡牌记录 - 纯本地离线版

## 📋 功能说明

- 纯前端，无需服务器
- 数据自动保存到浏览器 localStorage
- 图片自动裁剪为正方形
- 极简清爽界面
- 双击 HTML 即可使用

## 🚀 快速开始

直接双击打开 `index.html` 文件即可使用！

## 📂 文件结构

```
Cocktail/
├── index.html         # 网页界面
├── style.css          # 样式文件
├── script.js          # 前端逻辑
└── README.md          # 说明文档
```

## 📝 使用说明

### 添加记录

1. 点击「添加调酒卡牌」按钮
2. 上传图片（自动裁剪为正方形）
3. 填写酒吧名字、选择基酒
4. 添加风味标签
5. 填写评价和日期
6. 点击「提交保存」

### 数据保存

- **数据自动保存**：提交后立即保存到浏览器 localStorage
- **刷新不丢失**：刷新页面数据保持不变
- **无需联网**：完全离线使用

## ⚙️ 技术说明

- **存储方式**：localStorage
- **图片格式**：JPEG Base64 (500x500 裁剪)
- **排序方式**：按日期由新到旧

## 💡 提示

- 数据保存在浏览器中，清除浏览器数据会丢失记录
- 建议定期导出备份
